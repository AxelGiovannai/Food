# Food - Application de Recettes Android

## Description du Projet

Application Android permettant de consulter une liste de recettes de cuisine disponibles en ligne. L'application gère le réseau, propose une recherche/filtrage et persiste les recettes en cache local.

### Fonctionnalités :
- Écran de chargement avec logo
- Liste des recettes en cartes (image + titre)
- Barre de recherche et filtrage par catégories
- Écran détail avec liste d'ingrédients
- Gestion des erreurs réseau
- Cache local (base de données Room)

## Démarrage Rapide

### Prérequis
- Android Studio

### Installation

1. Extraire le fichier ZIP du projet
2. Ouvrir Android Studio
3. File → Open → Sélectionner le dossier extrait
4. Attendre la synchronisation Gradle automatique
5. Connecter un appareil ou démarrer un émulateur
6. Cliquer sur "Run" (Shift + F10)
